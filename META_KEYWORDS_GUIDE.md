# Meta Keywords Guide - How to Set Them Correctly

## What Are Meta Keywords?

Meta keywords help search engines understand what your page is about. They should be relevant, specific, and match the content.

## Best Practices

### 1. Use Relevant Keywords
✅ **Good:** Keywords that appear in your content
❌ **Bad:** Random keywords not related to content

### 2. Be Specific
✅ **Good:** "SSC CGL 2026 notification, SSC Combined Graduate Level exam"
❌ **Bad:** "jobs, government, exam"

### 3. Include Variations
- Main keyword
- Synonyms
- Common misspellings
- Related terms

### 4. Optimal Length
- **Recommended:** 5-15 keywords
- **Character limit:** 1000 characters (you'll see counter)
- **Separate with:** Commas

## Keyword Research Tips

### 1. Use Your Post Content
Look at your post and pick the most important terms:
- Job name
- Organization name
- Post count
- Exam name
- State/location
- Year

### 2. Think Like a Job Seeker
What would someone search for?
- "SSC CGL notification"
- "Railway jobs 2026"
- "UPSC admit card download"
- "Bank PO result"

### 3. Include Long-Tail Keywords
More specific = better targeting:
- ✅ "SSC CGL tier 1 admit card 2026"
- ❌ "admit card"

### 4. Add Location Keywords (if applicable)
- "Karnataka government jobs"
- "Delhi Police recruitment"
- "Maharashtra PSC notification"

### 5. Include Action Keywords
- "download"
- "apply online"
- "check result"
- "notification PDF"

## Common Mistakes to Avoid

### ❌ Keyword Stuffing
**Bad:**
```
jobs jobs jobs government jobs sarkari jobs govt jobs jobs 2026 jobs india jobs online
```
This looks spammy and doesn't help.

### ❌ Irrelevant Keywords
**Bad:** (For SSC post)
```
UPSC, Railway, Banking, Teaching, Police, Army, Navy
```
Only include if actually relevant.

### ❌ Too Generic
**Bad:**
```
jobs, exam, result, notification, government
```
Be specific!

### ❌ Too Few Keywords
**Bad:**
```
SSC CGL, exam
```
Add more relevant terms.

## Quick Formula

For any job post, include:

1. **Organization name** (SSC, UPSC, Railway, etc.)
2. **Exam/Post name** (CGL, NTPC, CSE, etc.)
3. **Year** (2026, 2027)
4. **Post type** (notification, admit card, result, syllabus)
5. **Post count** (if applicable)
6. **Location** (if state-specific)
7. **Related terms** (eligibility, exam date, application form)
8. **Action keywords** (apply online, download, check)

## Character Counter

When you type in the Meta Keywords field, you'll see:
```
Meta Keywords (280/1000 chars)
```

This shows:
- Current character count
- Maximum allowed (1000)

## Testing Your Keywords

After publishing, check if your keywords appear in:
1. Page title
2. Content body
3. Headings
4. Important links section

If a keyword doesn't appear in your content, remove it!

---

## Summary

✅ **DO:**
- Use 5-15 relevant keywords
- Include variations and synonyms
- Be specific to your post
- Separate with commas
- Stay under 1000 characters

❌ **DON'T:**
- Stuff keywords
- Use irrelevant terms
- Be too generic
- Repeat same keyword multiple times
- Use keywords not in your content

---

## Need Help?

If unsure, follow this simple rule:
**"If someone searches for this keyword, should they find my post?"**

If YES → Include it
If NO → Skip it
