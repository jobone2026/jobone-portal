  STEP-BY-STEP GUIDE
═══════════════════════════════════════════════════════════════════════════

1️⃣  LOGIN TO GITHUB
   - Go to: https://github.com
   - Login as: jobone2026

2️⃣  GO TO TOKEN SETTINGS
   - Click your profile picture (top right)
   - Click "Settings"
   - Scroll down to "Developer settings" (bottom left)
   - Click "Personal access tokens"
   - Click "Tokens (classic)"
   - Click "Generate new token" → "Generate new token (classic)"

3️⃣  CONFIGURE THE TOKEN
   
   Note: JobOne Deployment Token
   
   Expiration: Choose one:
   - 90 days (recommended)
   - No expiration (if you want it to work forever)
   
   Select scopes:
   ✅ repo (CHECK THIS BOX!)
      ├─ repo:status
      ├─ repo_deployment
      ├─ public_repo
      └─ repo:invite
   
   ⚠️  IMPORTANT: Make sure the 'repo' checkbox is CHECKED!
       This is the ONLY scope you need for pushing code.

4️⃣  GENERATE AND COPY TOKEN
   - Click "Generate token" (bottom of page)
   - You'll see a green token starting with "ghp_"
   - Click the copy icon to copy it
   - ⚠️  SAVE IT SOMEWHERE - you won't see it again!

5️⃣  USE THE TOKEN TO PUSH
   
   Open your terminal in this directory and run:
   
   # Update remote URL with your new token
   git remote set-url origin https://YOUR_NEW_TOKEN@github.com/jobone2026/jobone.git
   
   # Push to GitHub
   git push -u origin main
   
   Replace YOUR_NEW_TOKEN with the token you just copied.
