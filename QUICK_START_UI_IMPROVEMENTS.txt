💡 KEY FEATURES EXPLAINED
═══════════════════════════════════════════════════════════════════════

Urgency Indicator:
  • Automatically calculates days remaining until last date
  • Shows animated badge if ≤ 7 days remaining
  • Red gradient with pulse animation

Quick Apply Button:
  • Only shows on job posts (not on results, admit cards, etc.)
  • Finds "apply" link from important_links automatically
  • Falls back to first link if no apply link found
  • Floating position doesn't interfere with content

Toast Notifications:
  • Can be triggered from anywhere:
    showToast('Message here', 'success');
  • Types: success (green), error (red), info (blue), warning (yellow)
  • Auto-dismisses after 3 seconds
  • Manual close button available

Back to Top:
  • Only appears after scrolling 300px
  • Positioned left to avoid conflict with Quick Apply button
  • Smooth scroll animation (not instant jump)

🔧 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════

Changes not visible?
  → Clear caches: php artisan view:clear && php artisan cache:clear
  → Hard refresh browser: Ctrl+Shift+R (Windows) or Cmd+Shift+R (Mac)

Quick Apply button not showing?
  → Check if post type is 'job'
  → Verify important_links field has data
  → Clear view cache

Styles not applying?
  → Check Tailwind CSS compilation
  → Clear browser cache
  → Check console for errors
