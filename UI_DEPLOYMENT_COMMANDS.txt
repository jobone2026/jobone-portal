🔧 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════

Issue: Changes not visible
Solution:
  php artisan view:clear
  php artisan cache:clear
  Hard refresh browser (Ctrl+Shift+R)

Issue: Quick Apply button not showing
Solution:
  • Check if post type is 'job'
  • Verify important_links field has data
  • Clear view cache

Issue: Styles not applying
Solution:
  • Check if Tailwind CSS is compiled
  • Run: npm run build (if using Vite)
  • Clear browser cache

Issue: Components not found
Solution:
  • Verify all component files exist in resources/views/components/
  • Check file permissions
  • Clear view cache
