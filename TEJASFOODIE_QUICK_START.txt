TEJAS FOODIE DOMAIN - QUICK START
==================================

WHAT WAS CREATED:
-----------------
✓ Standalone landing page for tejasfoodie.store
✓ Completely separate from jobone.in and karnatakajob.one
✓ SSL-ready nginx configuration
✓ Automated setup script

FILES CREATED:
--------------
1. app/Http/Middleware/TejasFoodieMiddleware.php
2. public/tejasfoodie.html
3. tejasfoodie-nginx.conf
4. setup-tejasfoodie.sh
5. TEJASFOODIE_SETUP.md (detailed guide)

DEPLOYMENT STEPS:
-----------------

1. Point DNS to your server:
   A Record: tejasfoodie.store → Your Server IP
   A Record: www.tejasfoodie.store → Your Server IP

2. Wait 5-30 minutes for DNS propagation

3. Run the automated setup script:
   cd /var/www/govt-job-portal-new
   sudo bash setup-tejasfoodie.sh

4. Follow the prompts for SSL certificate

5. Done! Visit https://tejasfoodie.store

MANUAL SETUP (if you prefer):
------------------------------
See TEJASFOODIE_SETUP.md for step-by-step manual instructions

CUSTOMIZE LANDING PAGE:
-----------------------
Edit: public/tejasfoodie.html
Then: sudo systemctl reload nginx

IMPORTANT NOTES:
----------------
- This domain is 100% independent
- No database connection needed
- No Laravel routes needed
- Middleware intercepts all requests
- SSL auto-renews every 90 days
- Static HTML page (super fast)

TESTING:
--------
After setup, test these URLs:
- http://tejasfoodie.store (redirects to HTTPS)
- https://tejasfoodie.store (shows landing page)
- https://www.tejasfoodie.store (shows landing page)

TROUBLESHOOTING:
----------------
If issues occur:
1. Check nginx logs: sudo tail -f /var/log/nginx/tejasfoodie-error.log
2. Clear Laravel cache: php artisan cache:clear
3. Restart services: sudo systemctl restart nginx php8.2-fpm
